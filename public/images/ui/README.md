# UI Images

## Used By

Pages:

- global shared UI states

Components:

- buttons
- cards
- background layers
- future shared state visuals

## Purpose

This directory stores generic UI image assets that are not tied to a single business page.

Subdirectories:

- `button/`
- `card/`
- `background/`

## Naming

Use lowercase `kebab-case`.

Recommended patterns:

- `button-accent-soft.png`
- `card-shadow-paper.webp`
- `background-noise-soft.webp`

# Explore Inspiration Images

## Used By

Pages:

- `/explore`

Components:

- `InspirationCard`
- `InspirationExpandPanel`
- `SelectedInspirationSummary`

## Purpose

This directory stores the image groups for the Explore inspiration deck.

Subdirectories:

- `location/`
- `food/`
- `season/`
- `companion/`

## Required Image Groups

### `location`

- `beach`
- `mountain`
- `city`
- `forest`
- optional `desert`

### `food`

- `spicy`
- `seafood`
- `tea`
- `dessert`

### `season`

- `spring`
- `summer`
- `autumn`
- `winter`

### `companion`

- `couple`
- `family`
- `solo`
- `friends`

## Naming

Use lowercase `kebab-case`.

Recommended patterns:

- `beach-card.webp`
- `spicy-card.webp`
- `autumn-card.webp`
- `couple-card.webp`

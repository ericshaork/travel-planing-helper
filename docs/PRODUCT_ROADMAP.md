# 产品路线图：从 AI 行程草签到 AI 旅行工作台

## 1. 文档目的

这份路线图用来冻结版本边界，避免把多个方向揉进同一轮开发。

当前已完成：

- `v1.0`：一句话输入到完整行程生成
- `v1.1`：点击式积木编辑
- `v1.2`：Mobile Flow First
- `v1.2.5`：真实 AI 冒烟与 mobile 补丁
- `v1.3`：POI / Route / Weather / Route Insight 可信度增强
- `v1.4`：Desktop Workspace UI
- `v1.5`：真实地图基础版
- `v1.6`：登录 / 保存 / 我的行程 / 数据库

## 2. 当前版本结论

`v1.6` 已完成：

- 登录 / 登出
- 保存当前计划
- 我的行程列表
- 打开历史计划回到现有工作台
- 更新已保存计划
- 删除已保存计划
- Supabase `profiles` / `trip_plans`
- RLS 与基础安全边界

## 3. 后续路线

### `v1.7`

主题：UI / UX 大修

重点方向：

- 左侧导航遮挡修复
- 地图放大
- 天气展示旅行化优化
- marker / icon polish
- 欢迎页图片与整体视觉氛围优化
- Auth UX polish 评估

### `v1.8`

主题：内容生态

重点方向：

- 灵感卡
- 种子内容
- 路线模板
- Trae 内容流水线接入

### `v1.9`

主题：移动端工作台深度优化

重点方向：

- mobile 工作台结构补强
- 小屏编辑入口优化
- 移动端导航与信息密度调整

### `v2.0+`

主题：高级编辑与协作

重点方向：

- 地图选点
- 拖拽
- 自动重排
- 分享
- 协作

## 4. 当前仍不做

- 多人协作
- 公开分享
- 多版本历史
- 地图选点写回行程
- 拖拽编辑
- 自动重排

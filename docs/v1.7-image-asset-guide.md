## Contract Rules

- Prefer semantic filenames over numbered filenames for runtime card assets.
- Prefer `.png` runtime paths unless a component explicitly needs another format.
- Do not reference assets under `_source/` from runtime UI.
- Use resolver-based fallback chains instead of component-local image branching.
